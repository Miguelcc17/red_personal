# 06 - Testing

## Backend
- Utilizar `pytest` para pruebas unitarias.
- Ubicación: `backend/tests`.
- Ejecución: `pytest backend/tests`.

## Frontend
- Pruebas manuales de UI.
- Verificación de flujos CRUD completos desde el Dashboard.

## Verificación E2E
- Script de apoyo: `backend/tests/verify_functional.py`.
- Prueba conectividad de salud y flujos de creación/eliminación automáticos.
