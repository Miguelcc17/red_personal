# 02 - Estilo de Código

## Python (Backend)
- Seguir estándares **PEP 8**.
- Usar nombres descriptivos para variables y funciones (ej. `get_person_by_id`).
- Comentar funciones complejas.
- Mantener los Blueprints organizados por entidad.

## JavaScript/JSX (Frontend)
- Usar componentes funcionales y Hooks.
- Seguir la convención PascalCase para nombres de archivos de componentes.
- Utilizar Tailwind CSS para estilos consistentes.
- Manejar el estado local con `useState` y peticiones con `Axios`.
